
### Top 5 错误（快速参考）

| 错误 | 问题       | 快速检查                    |
| ---- | ---------- | --------------------------- |
| E001 | 异步未并行 | 是否用 `Promise.all()`?     |
| E002 | 轮询无超时 | 是否设置 `maxAttempts`?     |
| E003 | 错误未抛出 | `catch` 是否 `throw error`? |
| E004 | SQL未用CTE | JOIN 后过滤 → CTE 预过滤    |
| E007 | 资源泄漏   | 所有退出路径都清理?         |
